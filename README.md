# سياسة الخصوصية – تطبيق المعهد السعودي المتخصص العالي للتدريب

آخر تحديث: 25 يونيو 2026

## مقدمة

يرحب بكم المعهد السعودي المتخصص العالي للتدريب في تطبيقه الإلكتروني المخصص للطلاب. نحن نلتزم بحماية خصوصية مستخدمينا وضمان التعامل مع بياناتهم الشخصية بطريقة آمنة ومسؤولة. توضح هذه السياسة كيفية جمع واستخدام وحماية المعلومات عند استخدام التطبيق.

## المعلومات التي نقوم بجمعها

قد يقوم التطبيق بجمع بعض المعلومات اللازمة لتقديم الخدمات التعليمية، وتشمل:

* الاسم الكامل.
* رقم الجوال.
* البريد الإلكتروني.
* بيانات الحساب الدراسي.
* الفرع أو القسم المسجل به الطالب.
* الرسائل والإشعارات داخل التطبيق.
* الملفات والصور والمرفقات التي يقوم المستخدم برفعها.
* التسجيلات الصوتية التي يرسلها المستخدم عبر التطبيق.

## كيفية استخدام المعلومات

يتم استخدام المعلومات للأغراض التالية:

* إدارة حسابات المستخدمين.
* تقديم الخدمات التعليمية والتدريبية.
* التواصل مع الطلاب وإرسال الإشعارات المهمة.
* تحسين أداء التطبيق وتجربة المستخدم.
* تقديم الدعم الفني.
* حماية أمن النظام ومنع إساءة الاستخدام.

## مشاركة المعلومات

لا يتم بيع أو تأجير أو مشاركة المعلومات الشخصية مع أي طرف ثالث إلا في الحالات التالية:

* عند وجود متطلب قانوني أو أمر من الجهات المختصة.
* لحماية حقوق المعهد أو المستخدمين.
* مع مزودي الخدمات التقنية اللازمة لتشغيل التطبيق مع الالتزام بسرية البيانات.

## حماية البيانات

نلتزم باتخاذ الإجراءات التقنية والإدارية المناسبة لحماية البيانات من الوصول غير المصرح به أو التعديل أو الإفصاح أو الإتلاف.

## الاحتفاظ بالبيانات

يتم الاحتفاظ بالبيانات طالما كانت ضرورية لتقديم الخدمات التعليمية أو وفقًا للمتطلبات القانونية والتنظيمية المعمول بها.

## حقوق المستخدم

يحق للمستخدم:

* طلب تحديث بياناته الشخصية.
* طلب تصحيح البيانات غير الدقيقة.
* التواصل للاستفسار حول كيفية استخدام البيانات.

## خدمات الطرف الثالث

قد يحتوي التطبيق على روابط أو خدمات مقدمة من جهات خارجية، وتخضع تلك الخدمات لسياسات الخصوصية الخاصة بها.

## التعديلات على السياسة

يحتفظ المعهد بحق تعديل سياسة الخصوصية في أي وقت، وسيتم نشر أي تحديثات داخل التطبيق أو على هذه الصفحة.
